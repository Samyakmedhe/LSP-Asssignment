# LSP Assignment

A collection of Linux System Programming (LSP) assignments written in C.  
This repository includes multiple lab assignments covering essential file handling, directory traversal, and system-level programming concepts using POSIX APIs.


Each assignment folder contains the source code and sample input/output for a specific LSP task.
